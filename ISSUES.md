# oDoc — Issue Tracker

Thank you for taking the time to report a bug or request a feature!

## How to Report an Issue

Please use the [GitHub Issues](https://github.com/oddsifylabs/oDoc/issues) page to submit a report.  
When filing an issue, include as much detail as possible:

---

### 🐛 Bug Report Template

**Describe the bug**
A clear and concise description of what the bug is.

**To Reproduce**
Steps to reproduce the behavior:
1. Open oDoc
2. Load file type '...'
3. Perform action '...'
4. See error

**Expected behavior**
What you expected to happen.

**Actual behavior**
What actually happened.

**Screenshots**
If applicable, add screenshots to help explain the problem.

**Environment**
- OS: (e.g. macOS 14, Windows 11)
- Browser: (e.g. Chrome 124, Firefox 125, Safari 17)
- oDoc version: (e.g. v1.0)

---

### ✨ Feature Request Template

**Is your feature request related to a problem?**
A clear and concise description of what the problem is. E.g., "I'm always frustrated when..."

**Describe the solution you'd like**
A clear and concise description of what you want to happen.

**Alternatives considered**
Any alternative solutions or features you've considered.

**Additional context**
Any other context, mockups, or screenshots about the feature request.
